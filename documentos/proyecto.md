# Grupo AES - Sitio web en Wordpress

Grupo AES se dedica a Seguridad Privada, Limpieza Profesional y Suministro de Insumos para Empresas.

Adicionalmente se lanzará una marca propia de insumos de limpieza llamada Rumo.

Quieren un sitio web en Wordpress, responsivo, que tenga una página "hub" que concentre todas las anteriores, cada una con su propia página dedicada. El sitio debe ser amigable con SEO, por lo que se debe prestar atención al contenido y a la estructura del sitio. Cada página debe tener un H1 y al menos un H2.

Cada página dedicada debe tener un layout consistente con el de las otras, pero cada rubro debe tener su propia paleta de colores.
